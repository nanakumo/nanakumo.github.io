如何用统计软件进行简单的回归分析

  • Python
  • Stata
  • 经济统计
今天学了点什么呢

Python篇

package的准备

  1. pandas
  2. statsmodels
  3. openpyxl

读取文件

  1. 要指定好文件的路径
  2. 最好使用names=['x','y']来指定说明变量和目标变量。
import pandas as pd

file_pass = r"D:\python\data.xlsx"
df = pd.read_excel(file_pass, header=0, names=['cost', 'sales'])
import statsmodels.formula.api as smf

model = smf.ols('sales ~ cost', data=df)
result = model.fit()
print(result.summary())
