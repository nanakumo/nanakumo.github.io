# 前言

『語るなら未来を･･･』 是 欅坂 46 在 2016.8.10 发行的第二张单曲里的歌，而我是今年在玩音游ユニゾンエアー时才第一次听到。朗朗上口的曲调瞬间抓耳，标题也很喜欢，再加之我是一个比起过去更看重当下和未来的人，所以立刻就与歌词所传达出来的想法产生了共鸣。然后歌词里有一句是「把那些带到坟墓里去吧」，作为坟墓爱好者（咦？）就更喜欢这首歌了啊 wwwww（喂你在说什么啊 ——
